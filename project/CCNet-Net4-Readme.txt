CCNet-Net4
==========

This solution contains a number of extensions to CruiseControl.NET that use either .Net 4.0 or Visual Stduio 2012.

These extension rely on the binaries from the main solution. Please run build.bat (or build.sh) if any of the following references cannot be found:
* ThoughtWorks.CruiseControl.Core
* ThoughtWorks.CruiseControl.Remote

Notes: This binaries should be in the Build\Core folder
